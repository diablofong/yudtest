# Youtube playlist download test

## Required

1. nodejs
2. ffmpeg
3. php

## Project initalize

```
npm install
```

## Youtube playlist download

1. Add playlist code

```
let listCode = 'your download playlist code'
```

2. Run index.js download video

```
node index.js
```

## Convert mp4 to mp3

1. Run convert.php

```
php convert.php
```
